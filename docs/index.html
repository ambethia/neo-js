▸ getBlockCount()

Invokes the getblockcount rpc request to return the block height. This method will request the block height from the fastest active node with failover if a node is not provided. This method will update the blockHeight attribute on the node it is run on.

getBlockCount(): Promise<number>
Returns
Promise<number>: A promise returning the block count.
Example
node.getBlockCount()
return 1000000
